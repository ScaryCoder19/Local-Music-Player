# Music Player

A small, local music player you can run from the folder — no uploads, no accounts.

How to add music

1. Open `index.html` in your browser.
2. Click the little folder icon in the player controls, or drag files onto the playlist.
3. Pick one or more songs or a whole folder. The list fills up and playback starts.

What it supports

- Single audio files (mp3, m4a, wav, ogg, etc.)
- Entire folders of music (select a folder in the file dialog)

Quick notes

- Files are played locally in your browser — nothing leaves your machine.
- Tracks are ordered by filename/path when you load a folder.
- Playback moves to the next track automatically; use shuffle or repeat if you like.

If something doesn't start automatically, click Play once — some browsers block autoplay.
